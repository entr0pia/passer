# passer

A simple random password generator

## Usage

```bash
passer [length=12]
```
## Install

```bash
go get github.com/entr0pia/passer@latest
# go install github.com/entr0pia/passer@latest
```

## Character Set

- `ABCDEFGHJKLMNPQRSTUVWXYZ` 
    > Exclude `I` `O`
- `abcdefghijkmnopqrstuvwxyz`
    > Exclude `l`
- `0123456789`
- `@#$%^&*,./_-+=`
